# PathForge TD Gameplay Guide

本文档面向玩家、设计和测试人员，说明当前版本的玩法、操作、塔职责、敌人类型和验收关注点。

## 1. 游戏目标

玩家需要在每一关的网格地图上建造防御塔，消灭全部波次，避免敌人抵达核心。

核心体验是“塑造路径”：

- 地面敌人会根据当前地图和已建塔动态寻路。
- 玩家可以通过放塔迫使敌人绕路。
- 系统必须保证路径稳定、清晰，并禁止完全堵死任一入口。
- 飞行敌人不受地面迷宫影响，会直接接近核心。

## 2. 基础操作

| 操作 | 效果 |
| --- | --- |
| 点击塔卡 | 选择或取消一种塔 |
| 鼠标移动到地图 | 显示放置合法性和该塔作用范围 |
| 点击地图空格 | 尝试建造当前选中塔 |
| 点击已有塔 | 打开塔详情，可升级、出售、调整目标优先级 |
| 右键地图 | 取消当前建造选择 |
| 空格 | 建造阶段开始下一波；战斗阶段暂停/继续 |
| Esc | 打开暂停菜单 |
| 数字 1-8 | 快速选择 8 种防御塔 |
| Q/W/E | 切换 1x/2x/3x 速度 |

## 3. 游戏阶段

| 阶段 | 说明 |
| --- | --- |
| 菜单 | 主菜单、选关、设置、统计和成就 |
| 建造 | 可选塔、放塔、升级和开始下一波 |
| 战斗 | 敌人生成并移动，塔自动攻击 |
| 波次清理 | 当前波结束，等待下一波 |
| 胜利 | 战役关卡完成，显示评分和下一关入口 |
| 失败 | 生命归零，显示重开和返回入口 |

## 4. 防御塔职责

每张塔卡必须让玩家立即理解“这座塔解决什么问题”。当前 8 种塔的定位如下：

| 塔 | 快捷键 | 定位 | 主要用途 |
| --- | --- | --- | --- |
| 箭塔 | 1 | 单体 | 前期稳定输出，清理普通地面敌人 |
| 炮塔 | 2 | 范围 | 处理密集地面敌人，和减速效果配合最好 |
| 冰塔 | 3 | 减速 | 放在转角和长路，延长敌人受击时间 |
| 雷塔 | 4 | 对空 | 同时打地面和飞行，适合飞行波前准备 |
| 毒塔 | 5 | 毒伤 | 持续伤害，克制高血量、盾兵和 Boss |
| 狙击塔 | 6 | 狙杀 | 超远距离高单发，点杀强敌和漏怪 |
| 增益塔 | 7 | 辅助 | 放在塔群中心，提高周边塔输出效率 |
| 兵营 | 8 | 控制 | 核心前、转角和汇合点控制地面敌人 |

## 5. 塔说明标准

每个塔的 UI 说明必须覆盖：

- 定位：单体、范围、减速、对空、毒伤、狙杀、辅助或控制。
- 目标：地面、飞行或两者都能攻击。
- 伤害：物理、火、冰、雷、毒或真实伤害。
- 特殊效果：例如减速、眩晕、溅射、中毒、辅助 Buff。
- 推荐场景：转角、长路、塔群中心、核心前、飞行波等。
- 克制对象：普通小怪、集群、高血量、护甲、飞行或 Boss。

文案来源优先使用 `TowerConfig.role / description / special / usage`。

## 6. 敌人类型

| 敌人 | 特点 | 推荐应对 |
| --- | --- | --- |
| 史莱姆 | 基础低血量敌人 | 箭塔、炮塔 |
| 狼 | 高速地面敌人 | 冰塔、箭塔、核心前补火力 |
| 兽人 | 血量较高且有护甲 | 毒塔、炮塔、狙击塔 |
| 飞行者 | 飞行，不受地面迷宫影响 | 雷塔、毒塔、狙击塔 |
| 盾兵 | 高护甲慢速单位 | 毒塔、炮塔、长路线 |
| 幽灵 | 飞行且有魔抗 | 雷塔数量不要过少，配合狙击塔 |
| 火元素 | 高魔抗地面单位 | 物理塔、路径拉长 |
| 治疗者 | 支援单位 | 狙击塔优先点杀 |
| 自爆怪 | 接近后影响塔群 | 冰塔拖延，狙击塔提前击杀 |
| 迅捷刺客 | 高速且可冲刺 | 减速、控制、核心前兜底 |
| 森林巨兽 | 召唤型 Boss | 毒塔、狙击塔、增益塔 |
| 熔岩领主 | 地形压迫型 Boss | 分散塔群，保持长路径输出 |
| 天空暴君 | 飞行 Boss | 雷塔、毒塔、狙击塔集中防空 |

## 7. 关卡节奏

当前战役包含 24 个关卡，从 `1-1` 到 `8-3`：

- 第 1 章：基础规则、路线塑形、飞行敌人引入。
- 第 2 章：地形、双入口和更高压力。
- 第 3-8 章：复用高级关卡模板，逐步提高起始金币、敌人复杂度、波次数和 Boss 压力。

关卡设计要求：

- 入口和核心必须固定、明显、不可移动。
- 可建造区、主路、障碍、森林、水和熔岩都要有明确用途。
- 地图不能有大量无意义空地。
- 玩家每次放塔后应能看到路径变化，但不应看到地图格子本身移动。

## 8. 评分与进度

战役胜利后按剩余生命比例给星：

| 条件 | 星级 |
| --- | --- |
| 剩余生命 >= 80% | 3 星 |
| 剩余生命 >= 40% | 2 星 |
| 胜利但低于 40% | 1 星 |
| 失败 | 0 星 |

胜利后会保存关卡最高星数，并在结算界面提供下一关入口。无尽模式记录最高波次。

## 9. 新手可读性要求

玩家不应依靠试错才能理解核心信息：

- 塔卡显示名称、价格、定位和简短职责。
- 选中塔后，地图悬停显示范围和能否放置。
- 下一波预览显示即将出现的敌人和奖励。
- 飞行波次前，界面或路线预览要提醒防空压力。
- 放塔失败时应反馈原因，同时保留选中的塔和说明面板。

## 10. 手测路径

一次完整玩法手测建议覆盖：

1. 从主菜单进入 `1-1`。
2. 逐个点击 8 种塔，确认说明面板可读。
3. 选择塔并悬停地图，确认范围圈和合法性提示显示。
4. 在可建造格放塔，确认金币扣除、路径刷新、UI 不消失。
5. 尝试堵死路径，确认放塔失败且选中状态保留。
6. 开始波次，确认敌人移动、塔攻击、生命和金币更新。
7. 通关后点击下一关，确认进入下一个关卡。
